
=========
BaseImage
=========

Image class used by ``pygame-menu``.


Drawing image modes
-------------------

=====================================================  ===========================================
Drawing mode                                           Description
=====================================================  ===========================================
:py:data:`pygame_menu.baseimage.IMAGE_MODE_CENTER`     Image is drawn at the center of the surface
:py:data:`pygame_menu.baseimage.IMAGE_MODE_FILL`       Image fills the surface
:py:data:`pygame_menu.baseimage.IMAGE_MODE_REPEAT_X`   Image repeats on X axis
:py:data:`pygame_menu.baseimage.IMAGE_MODE_REPEAT_XY`  Image repeats on both axis
:py:data:`pygame_menu.baseimage.IMAGE_MODE_REPEAT_Y`   Image repeats on Y axis
:py:data:`pygame_menu.baseimage.IMAGE_MODE_SIMPLE`     Just place the image in the top/left corner
=====================================================  ===========================================
